## 🚀 사용 방법

### 빠른 시작 (5분 안에 완료)

1. **Google AI Studio 접속**
   - URL: https://aistudio.google.com
   - 구글 계정 로그인

2. **Gems 생성**
   - "Create new" → "Gem" 선택
   - 이름: `노인보호 상담일지 자동 생성기`

3. **지침서 복사**
   - `GEMINI_GEMS_INSTRUCTION.md` 파일 전체 복사
   - "Instructions" 필드에 붙여넣기

4. **테스트**
   - 샘플 녹취록 입력
   - JSON 상담일지 자동 생성 확인

5. **저장 및 사용**
   - "Save Gem" 클릭
   - 이후 언제든지 재사용 가능!

## 🆘 문제 해결 가이드

### 문제 1: JSON 형식 깨짐
- **증상**: ```json 블록 포함, 괄호 누락
- **해결**: 출력에서 ```json과 ``` 제거, 괄호 수동 추가

### 문제 2: "미입력" 과다
- **증상**: 50% 이상 필드가 "미입력"
- **해결**: 녹취록에 더 많은 맥락 정보 제공

### 문제 3: 환각 발생
- **증상**: 녹취록에 없는 내용 추가
- **해결**: "이 내용은 녹취록에 없었습니다" 피드백

### 문제 4: 할당량 초과
- **증상**: "429 Too Many Requests" 오류
- **해결**: 다음 날까지 대기 또는 유료 API 사용
