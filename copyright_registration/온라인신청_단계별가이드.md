## 📝 Step 1: 회원가입

### 1-1. 사이트 접속
```
https://www.cros.or.kr
```

### 1-2. 회원가입 클릭
```
홈페이지 우측 상단
→ "회원가입" 버튼 클릭
```

### 1-3. 약관 동의
```
- [V] 이용약관 동의 (필수)
- [V] 개인정보 수집 및 이용 동의 (필수)
- [ ] 마케팅 정보 수신 동의 (선택)
→ "다음" 버튼
```

### 1-4. 본인 인증
```
방법 선택:
1. 휴대폰 인증 (추천)
2. 아이핀 인증
3. 공동인증서 (구 공인인증서)

→ 휴대폰 인증 권장 (가장 간단)
```

### 1-5. 회원정보 입력
```
아이디: [5-20자 영문/숫자]
비밀번호: [8자 이상, 영문+숫자+특수문자]
비밀번호 확인: [동일하게 입력]
이름: [실명]
생년월일: [YYYYMMDD]
휴대폰: [010-1234-5678]
이메일: [your@email.com]

→ "가입하기" 클릭
```

### 1-6. 이메일 인증
```
가입한 이메일 확인
→ 인증 메일 수신
→ 인증 링크 클릭
→ 회원가입 완료!
```

## 📎 Step 4: 서류 업로드

### 4-1. 첨부 서류 안내

시스템에서 요구하는 서류:

```
1. 프로그램 설명서 (필수)
2. 소스코드 일부 (필수)
3. 화면 캡처 (권장)
4. 기타 참고 자료 (선택)
```

### 4-2. 파일 준비

#### 준비해야 할 파일:

```
1. 프로그램설명서.pdf (15페이지)
2. 소스코드_처음30페이지.pdf
3. 소스코드_마지막30페이지.pdf
4. 화면캡처.zip (10장의 PNG/JPG를 ZIP으로 압축)
5. 신분증사본.pdf
```

#### PDF 변환 방법:

**Markdown → PDF 변환**
- 온라인: https://www.markdowntopdf.com/
- 또는: Word에 복사 → PDF로 저장

**이미지 → ZIP 압축**
```
화면캡처 10장 선택
→ 우클릭 → "압축" 선택
→ 파일명: 화면캡처.zip
```

### 4-3. 업로드

```
첨부 파일 1: 프로그램 설명서
→ [파일 선택] → 프로그램설명서.pdf 업로드

첨부 파일 2: 소스코드 (처음 30페이지)
→ [파일 선택] → 소스코드_처음30페이지.pdf 업로드

첨부 파일 3: 소스코드 (마지막 30페이지)
→ [파일 선택] → 소스코드_마지막30페이지.pdf 업로드

첨부 파일 4: 화면 캡처
→ [파일 선택] → 화면캡처.zip 업로드

첨부 파일 5: 신분증 사본
→ [파일 선택] → 신분증사본.pdf 업로드
```

**파일 크기 제한**:
- 개별 파일: 최대 50MB
- 전체 합계: 최대 200MB

## 💳 Step 5: 수수료 납부

### 5-1. 수수료 확인
```
컴퓨터프로그램 저작권 등록: 40,000원
프로그램 설명서: 15,000원
─────────────────────────────
합계: 55,000원
```

### 5-2. 결제 방법 선택

#### 신용카드 (즉시 결제)
```
→ "신용카드" 선택
→ 카드 정보 입력
→ "결제하기" 클릭
→ 즉시 승인
```

#### 계좌이체
```
→ "계좌이체" 선택
→ 은행 선택
→ 공인인증서 인증
→ 계좌번호 입력
→ 이체 승인
```

#### 가상계좌
```
→ "가상계좌" 선택
→ 은행 선택
→ 가상계좌 번호 발급
→ 해당 계좌로 입금 (24시간 내)

예) 국민은행 123-456-789012
    입금자명: [귀하의 성명]
    금액: 55,000원
```

### 5-3. 결제 완료
```
결제 완료 메시지 확인
→ 영수증 자동 발급
→ 이메일로 영수증 수신
```

## 🆘 문제 해결 (FAQ)

### Q1. 회원가입 시 본인 인증 실패
```
해결:
- 휴대폰 번호 정확히 입력
- 이통사 선택 확인
- 브라우저 쿠키 삭제 후 재시도
- 다른 인증 방법 시도 (아이핀 등)
```

### Q2. 파일 업로드 실패
```
해결:
- 파일 크기 확인 (50MB 이하)
- 파일 형식 확인 (PDF, ZIP 등)
- 브라우저 변경 (Chrome 권장)
- 인터넷 속도 확인
```

### Q3. 결제 오류
```
해결:
- 카드 한도 확인
- 다른 결제 수단 시도
- 브라우저 팝업 차단 해제
- 고객센터 문의 (1800-5455)
```

### Q4. 추가 서류 요청
```
대응:
- 이메일 확인 (요청 내용)
- 빠르게 준비 (3일 이내 권장)
- 나의 신청 내역에서 업로드
- 제출 완료 문자 확인
```

### Q5. 반려 통지
```
대응:
- 반려 사유 확인
- 서류 보완
- 재신청 (수수료 재납부 필요)
→ 최초 신청 시 꼼꼼히 준비하여 반려 방지!
```
