## 🚀 빠른 시작 가이드

### Step 1: 개인정보 입력 (5분)

다음 파일을 열어서 `[  ]` 부분을 채우세요:

**📄 documents/저작권등록신청서.md**
```
저작자 성명: [귀하의 성명을 입력하세요]
생년월일: [YYYYMMDD 형식으로 입력]
주소: [귀하의 주소를 입력하세요]
전화번호: [귀하의 전화번호]
이메일: [귀하의 이메일]
```

**📄 documents/프로그램설명서.md**
```
문의처 섹션 (마지막 부분):
- 성명: [귀하의 성명]
- 이메일: [귀하의 이메일]
- 전화: [귀하의 전화번호]
```

### Step 2: 화면 캡처 (10분)

다음 페이지들을 방문하여 화면 캡처:

1. https://casenetai.kr/ (메인 페이지)
2. https://casenetai.kr/login.html (로그인)
3. https://casenetai.kr/elderly-protection.html (상담일지)
4. https://casenetai.kr/statement-recording.html (진술서)
5. https://casenetai.kr/anonymization.html (익명화)
6. https://casenetai.kr/welfare-facilities.html (복지시설)
7. https://casenetai.kr/admin-setup.html (관리자 설정)
8. 기타 주요 기능 화면 3장

**저장 위치**: `screenshots/` 폴더에 저장
**파일명**: `화면01_메인페이지.png`, `화면02_로그인.png` 등

### Step 3: PDF 변환 (5분)

모든 `.md` 파일을 PDF로 변환:

**온라인 변환기 사용** (추천)
- https://www.markdowntopdf.com/
- 파일 업로드 → PDF 다운로드

**또는 Word 사용**
- Markdown 내용 복사 → Word에 붙여넣기 → PDF로 저장

**변환할 파일**:
1. 저작권등록신청서.md → 저작권등록신청서.pdf
2. 프로그램설명서.md → 프로그램설명서.pdf
3. 소스코드_처음30페이지.txt → 소스코드_처음30페이지.pdf
4. 소스코드_마지막30페이지.txt → 소스코드_마지막30페이지.pdf

### Step 4: 신분증 사본 준비 (2분)

신분증 사진 촬영 또는 스캔:
- 주민등록증 또는 운전면허증
- 뒷면 주민번호는 가려도 됨
- PDF로 저장: `신분증사본.pdf`

### Step 5: 온라인 신청 (10분)

**📄 온라인신청_단계별가이드.md** 파일을 따라서 진행:

1. https://www.cros.or.kr 접속
2. 회원가입 및 로그인
3. 저작권 등록 신청
4. 서류 업로드
5. 수수료 납부 (55,000원)
6. 완료!
