## ✅ 준비 서류

### 1. 필수 서류

- [ ] **저작권 등록신청서**
  - 파일: `documents/저작권등록신청서.md`
  - 작성 항목: 저작자 성명, 생년월일, 주소, 연락처 입력 필요
  - 서명란에 실제 서명 필요

- [ ] **프로그램 설명서**
  - 파일: `documents/프로그램설명서.md`
  - 내용: 기능, 기술 스택, 시스템 구성 포함
  - 분량: 약 15페이지

- [ ] **소스코드 일부 (60페이지)**
  - 처음 30페이지: `source_code/소스코드_처음30페이지.txt`
  - 마지막 30페이지: `source_code/소스코드_마지막30페이지.txt`
  - 형식: 텍스트 파일 또는 PDF

- [ ] **화면 캡처** (10장 이상)
  - 메인 페이지
  - 로그인 화면
  - 상담일지 생성 화면
  - 진술서 작성 화면
  - 문서 익명화 화면
  - 복지시설 조회 화면
  - 관리자 페이지
  - 크레딧 관리 화면

### 2. 신분 증명

- [ ] **신분증 사본**
  - 주민등록증 또는 운전면허증
  - 개인정보 일부 가림 가능 (주민번호 뒷자리 등)

### 3. 수수료

- [ ] **납부 확인**
  - 컴퓨터프로그램 저작권 등록: 40,000원
  - 프로그램 설명서: 15,000원
  - **합계: 55,000원**

## 🌐 온라인 제출 방법

### 접속

**한국저작권위원회 저작권 온라인 등록시스템**
- URL: https://www.cros.or.kr
- 회원가입 필요

### 절차

#### 1단계: 회원가입 및 로그인
```
https://www.cros.or.kr
→ 회원가입
→ 이메일 인증
→ 로그인
```

#### 2단계: 등록 신청
```
메뉴: 저작권 등록 신청
→ 컴퓨터프로그램저작물 선택
→ 저작권 등록신청서 작성
→ 온라인 양식에 정보 입력
```

#### 3단계: 서류 업로드
```
첨부 서류 업로드:
- 프로그램 설명서 (PDF)
- 소스코드 (처음 30 + 마지막 30페이지)
- 화면 캡처 (ZIP 압축)
- 신분증 사본
```

#### 4단계: 수수료 납부
```
결제 방법 선택:
- 신용카드
- 계좌이체
- 가상계좌

금액: 55,000원
```

#### 5단계: 접수 완료
```
접수번호 확인
→ 이메일로 접수증 수신
→ 심사 대기 (약 2-3주)
```
